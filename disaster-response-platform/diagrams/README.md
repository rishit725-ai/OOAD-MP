# Diagrams — Mermaid Source Files

## How to View These Diagrams in draw.io

1. Open **draw.io** (app.diagrams.net or desktop app)
2. Click **Extras → Edit Diagram**
3. In the dialog, click the format dropdown (shows "XML" by default)
4. Select **"Mermaid"**
5. Paste the Mermaid code from the `.md` file below
6. Click **OK**

The diagram will be rendered and can be saved as PNG/SVG.
